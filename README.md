A clean, semantic HTML5 professional profile page built to demonstrate structured markup, accessible content, and a validated contact form.



- Proper semantic HTML structure (header, nav, main, section, article, footer)
- Internal jump navigation links
- External social link
- Skills list (unordered list)
- Project section using article elements
- Responsive, non-distorted profile image
- Fully labeled and validated contact form
- Accessible alt text for images
